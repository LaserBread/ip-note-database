# IP Note Database
This is a simple database project meant to be used with an Android app. It stores IP addresses from the app in a database. The only reason I have it set up like this is because the requirements of the final.

> [!NOTE]
> This project is part of my final for CS 492. As such, it's far from production worthy. I recommend you use this only on your own personal network or behind a VPN.

## Setup
1. Clone the file onto your machine.
2. Set up the virtual environment and install everything in requirements.txt
3. Make sure you have MariaDB installed and the correct credentials.
4. Run `setup.sql` in the database you want to hold this data in
5. Run `python3 app.py`
